# 📝 Fullstack Notes Architecture (Monorepo)

Proyecto avanzado que implementa un sistema de gestión de notas utilizando una arquitectura de **Monorepo**. Este diseño permite gestionar tanto el Backend (API) como el Frontend (Cliente) en un único repositorio unificado, facilitando la consistencia del código.

## 🏗️ Arquitectura del Proyecto
- **API (Server):** Construida con Node.js y Express, encargada de la lógica de negocio y persistencia.
- **App (Client):** Interfaz de usuario moderna desarrollada en React para una experiencia fluida.

## 🛠️ Tecnologías y Conceptos
- **Arquitectura:** Monorepo.
- **Frontend:** React.js.
- **Backend:** Node.js / Express.
- **Gestión de Estado:** Manejo de datos asíncronos y consumo de API REST.
